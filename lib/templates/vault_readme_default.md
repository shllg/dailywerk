# DailyWerk Configuration

This folder contains configuration that controls how your DailyWerk AI assistant
interacts with this vault.

## Files

- `vault-guide.md` contains the structure rules the agent follows
- `vault-analysis.md` contains generated analysis for imported vaults

## Important

- Do not delete this folder; DailyWerk will recreate it
- Edits to `vault-guide.md` take effect on the next agent interaction
- This folder is portable and safe to sync with the rest of the vault
